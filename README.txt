Installation

```bash

git clone https://github.com/madmoo-Pi/mandelmind.git
cd mandelmind

# Install dependencies
pip install -r requirements.txt

# Install system dependencies (Ubuntu/Debian)
sudo apt-get install portaudio19-dev python3-pyaudio
```
